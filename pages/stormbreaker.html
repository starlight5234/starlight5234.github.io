Who are we?

We here at Team StormBreaker, aim for a kernel with fine tuned balance between performance, battery & usage stability while reaching Android's requirements & being up-to-date with LTS release. It has an easy installation process thanks to AnyKernel3 (by osm0sis) which lets us support Custom ROMs.

Features

All the basic features(every bug is an intended feature btw) such as touch, audio, fingerprint, camera, and more work just fine. However Xiaomi is notorious these days so if your touchscreen doesn't work, I won't be amused. You can fill the contact form and I'll provide assistance for you if it is possible.

  • Based on LA.UM.9.15.r1-02000-KAMORTA.0
  • Supports Android 11
  • Enable Power Efficient Workqueue
  • Faster & better deepsleep
  • Support ExFat (for sdcards)
  • Updated wlan & techpack drivers from LA.UM.9.15.r1-02000-KAMORTA.0

Installation

  • Make sure you have a custom recovery like Orangefox/TWRP
  • Download the kernel zip from downloads section
  • Reboot to recovery
  • Backup your current kernel a.k.a Boot Partition
  • Install the kernel zip file without wiping any partitions
  • Reboot back to system
  • Enjoy! :D

NOTE : Give the kernel atleast a 2-3 day usage cycle before complaining because EAS uses precision based on CPU frequency scaling, task placement on CPU cores, and task migration between cores.
