StormBreaker Kernel for Redmi 5 Plus

codename vince
Co-maintainer: Saalim

About us:

We here at Team StormBreaker, aim for a kernel with fine tuned balance between performance, battery & usage stability while reaching Android's requirements & being up-to-date with LTS release. It has an easy installation process thanks to AnyKernel3 (by osm0sis which lets us support Custom ROMs.

Since this kernel is 4.9 it's based on Energy Aware Scheduling (EAS) known to use the most efficienct CPU for required task whilst aiming to improve energy efficient system operation, in short yielding more Screen On Time (SOT) than LTS kernels without EAS. The kernel is always compiled with Latest GCC compiler by Arter97

Features:

  • Rebased to LA.UM.9.6.2.r1-02800-89xx.0
  • Supports Android 10 and 11
  • Enable vDSO 32-bit (for almost 0.4% better battery according to Google)
  • Upstream lz4 from 4.14 LTS
  • Enable Power Efficient Workqueue
  • Add Kcal Implementation (for adjusting screen colors)
  • Fix Gcam focus & slow motion bug
  • Scheduler changes from 4.14 (increasing reliability of EAS)
  • Faster & better deepsleep
  • Disable I/O stat accounting (reduces overhead)
  • Add Vibration control
  • Add Sound control
  • Support ExFat (for sdcards)
  • Disable unneccessary debugging (reduces overhead)
  • Add CPU Input & DevFreq Boost
  • Remove CAF's hispeed functionality (kdragon: reduce useless overhead in schedutil)
  • Reduce power consumption of sensors and camera
  • Remove unneccessary wakelocks
  • Implement USB 3.0 Fast Charging
  • Update wlan and audio drivers from LA.UM.9.6.2.r1-02800-89xx.0
  • Drop Core control (Already dropped by CAF for msm8953, hence redundant code)

Installation

- Make sure you have a custom recovery like Orangefox
- Download the kernel zip from downloads section
- Reboot to recovery
- Backup from current kernel (Boot Partition)
- Install the kernel zip file without wiping any partitions
- Reboot back to system
- Enjoy! :D
